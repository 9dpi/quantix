# AI SMART FORECAST COMMERCIAL EDITION

## 1. Tổng quan hệ thống (System Overview)
- **Mô tả:** Hệ thống dự báo chứng khoán tự động hóa 100%, sử dụng thuật toán AI để phân tích biến động thị trường và đưa ra tín hiệu giao dịch.
- **Trạng thái hiện tại:** V1.0 ổn định trên VN30.
- **Hạ tầng:** GitHub Actions (Backend/Cron) + GitHub Pages (Frontend).
- **Mục tiêu:** Low cost, High availability.

## 2. Các tính năng thương mại (Commercial Features)
- **The Gatekeeper:** Landing page chuyên nghiệp, thu hút.
- **User Login System:** Ẩn dữ liệu tín hiệu (Blur/Hide) với khách vãng lai. Chỉ hiển thị cho Member.
- **Lead Collection:** Thu thập email.
- **Global Scaling:** Support NYSE, NASDAQ, Crypto, Forex.

## 3. Giải pháp kỹ thuật (Technical Solutions)
- **Security:** Chống F12, Right-Click (Client-side protection).
- **Optimization:** Data-fetching chỉ dành cho user đã đăng nhập.
- **Giao diện:** Song ngữ (Anh/Việt), Dark Mode/Neon Style.
